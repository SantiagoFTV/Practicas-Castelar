# Proyecto El Puente - Backend - Sprint 1

## Estructura
- Node.js + Express
- PostgreSQL
- Autenticación JWT
- Variables de entorno en `.env`

## Scripts
- `npm install` para instalar dependencias
- `npm start` para iniciar el servidor

## Endpoints principales
- `POST /api/auth/register` — Registro de usuario
- `POST /api/auth/login` — Login de usuario (devuelve JWT)

## Base de datos
Ver `models.sql` para la estructura inicial de usuarios.
