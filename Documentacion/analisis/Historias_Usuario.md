# Historias de Usuario

## Historia 1: Registro de usuario
Como visitante,
quiero poder crear una cuenta con mi correo, usuario y contraseña,
para acceder a la plataforma y guardar mi progreso.

## Historia 2: Inicio de sesión
Como usuario registrado,
quiero poder iniciar sesión con mi correo y contraseña,
para acceder a mi perfil y funcionalidades personalizadas.

## Historia 3: Recuperar contraseña
Como usuario,
quiero poder recuperar mi contraseña si la olvido,
para poder volver a acceder a mi cuenta.

## Historia 4: Subir imagen y comentarios positivos
Como usuario,
quiero poder subir una imagen y escribir comentarios positivos,
para compartir mi experiencia y motivar a otros.

## Historia 5: Ver mensajes de éxito y error
Como usuario,
quiero ver mensajes claros cuando registro, inicio sesión o subo información,
para saber si la acción fue exitosa o si hubo algún problema.
