## Sprint 1 – Base del Sistema (Autenticación y Estructura)

### Objetivo
Tener la aplicación funcionando con registro e inicio de sesión seguro.

### Funcionalidades
- Registro de usuario
- Login con autenticación JWT
- Encriptación de contraseñas
- Estructura base de base de datos
- Diseño básico inicial (estructura visual calmada)

### Entregable
Usuario puede registrarse e iniciar sesión correctamente.
